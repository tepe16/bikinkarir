# Menjadi Frontend Developer

Sebelum melangkah lebih jauh ke tips dan trik nya, sebelumnya kita harus tau dulu apa sih frontend developer ? nahh ini saya jelaskan , frontend developer adalah pengembang website yang menggunakan baris kode HTML, CSS, dan JavaScript untuk menghasilkan website dengan tampilan yang menarik, lebih tepatnya adalah tampilan website nya dengan tujuan agar di webstite nya tersebut kelihatan menarik bagi penggunanya.

Berkaitan dengan frontend developer, pada tips dan trik kali ini adalah passion seorang frontend developer adalah menyukai desain web, selain itu di dunia ilmu software developer khususnya frontend hal yg mendasar adalah mengetahuhi dasar-dasar frontend dari html,css dan javascript pada saat matakuliah, html dan css adalah materi yg di ajarkan waktu duduk di bangku kuliah, sedangkan untuk javascript sendiri dengan cara explore belajar melalui situs di internet

Bagaimana cara mendapatkan pekerjaan sebagai frontend developer ?
1. Mengetahui dasar-dasar dari html,css dan javascript.
2. Konsisten.
3. Mau belajar dan explore.
4. Buat Project kecil untuk mengasah skill.

# Menjadi Frontend Developer

Sebelum melangkah lebih jauh ke tips dan trik nya, sebelumnya kita harus tau dulu apa sih backend developer ? nahh ini saya jelaskan , backend developer adalah seseorang yang akan mengatur hal-hal yang berkaitan dengan server dan database. Back end developer menjadi sosok yang akan memastikan keamanan website bekerja dengan baik dan sangat aman tanpa bisa ditembus oleh hacker untuk mencuri data di website.

Berkaitan dengan backend developer, pada tips dan trik kali ini adalah passion seorang backend developer adalah struktur data, database,merancang alur website agar proses berjalan sesuia keinginan, selain itu di dunia ilmu software developer khususnya backend hal yg mendasar adalah mengetahuhi dasar-dasar backend contohnya php,go,NodeJs dan lain-lain.

Bagaimana cara mendapatkan pekerjaan sebagai frontend developer ?
1. Mengetahui dasar-dasar dari backend seperti Golang,php, NodeJs dll.
2. Konsisten.
3. Mau belajar dan explore.
4. Buat Project kecil untuk mengasah skill.
